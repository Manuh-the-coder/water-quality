Help Center

How to Use the System

  • Login to the system with your username and password.
  • Once logged in, navigate to the dashboard to view real-time data.
  • If a parameter exceeds the threshold, you will see a warning displayed on the dashboard.
  • You can view detailed monitoring reports from the 'Monitoring Report' section.

Troubleshooting

  • If you are unable to log in, please ensure that your username and password are correct.
  • If you receive incorrect sensor data or warnings, try refreshing the page or simulating new data.
  • For any other issues, please contact support.
